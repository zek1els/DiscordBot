      Custom commands

      Users type !command in chat (e.g. !hug @max). The bot replies with your template. Use placeholders: {author} (who ran it), {target} (first @mention), {author_username}, {target_username}, {args} (rest of message).

      If nothing happens when you type !command: In the Discord Developer Portal open your app → Bot → under Privileged Gateway Intents turn on MESSAGE CONTENT INTENT, then save. Restart the bot if it was already running.

                  Jail system

                  Remove server access from a user and assign them a "criminal" role.

                  Setup: Run /jail-setup in Discord and pick the member role + criminal role.
                  Assign all: /jail-assign-all gives the member role to every existing member.
                  Usage: !jail @user to jail, !unjail @user to release.
                  Auto-role: New members automatically get the member role.
